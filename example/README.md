### minimal setup

```
npm install reaqt --save
```

&nbsp;

### folder structure

    .
    ├── pages         (top level routes)
    ├── components    (organize them as you like)
    └── static        (images, fonts, that kind of stuff)

&nbsp;

### dev mode

```
npm run dev

> Running on http://localhost:3000
```

&nbsp;

### ready for production?

```
npm start
```

that's all you need
